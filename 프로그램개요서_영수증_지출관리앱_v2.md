# 🧾 영수증 지출 관리 앱 (Receipt Expense Tracker)
### 프로그램 개요서 | Ver 2.0 | 2025

---

## 1. 프로그램 목적

영수증(이미지/PDF)을 업로드하면 AI가 자동으로 내용을 인식하여 지출 데이터로 변환하고,
사용자가 소비 내역을 손쉽게 조회·관리할 수 있는 경량 웹 애플리케이션입니다.
수동 입력 없이 영수증 촬영만으로 가계부를 자동화하는 것을 목표로 합니다.

---

## 2. 타겟 사용자

- 영수증 촬영만으로 지출을 간편하게 기록하고 싶은 개인 사용자
- 복잡한 가계부 앱 없이 소비 패턴을 파악하고 싶은 사용자
- AI 기반 OCR 기술을 활용한 사이드 프로젝트에 관심 있는 개발자

---

## 3. 핵심 기능

| # | 기능 | 설명 |
|---|------|------|
| 1 | 영수증 업로드 | 이미지(JPG, PNG) 및 PDF 파일 업로드 |
| 2 | AI 자동 파싱 | 가게명, 날짜, 품목, 금액 자동 추출 |
| 3 | 지출 내역 조회 | 저장된 지출 목록 카드형 리스트 표시 |
| 4 | 지출 합계 대시보드 | 총 지출 및 기간별 소비 요약 |
| 5 | 내역 수정 및 삭제 | OCR 오류 수정 및 잘못된 항목 삭제 |

---

## 4. 기술 방향 및 운영 환경

| 항목 | 내용 |
|------|------|
| 개발 기간 | 1일 (단기 스프린트) |
| 프론트엔드 | React 기반 웹 애플리케이션 |
| 백엔드 | Python 기반 REST API 서버 |
| AI 엔진 | Upstage Vision LLM (OCR 및 데이터 추출) |
| 데이터 저장 | DB 미사용 — JSON 파일 기반 경량 운영 |
| 배포 환경 | Vercel (GitHub 연동 자동 배포) |

---

## 5. 시스템 아키텍처

### 전체 흐름

```
[사용자 브라우저]
  React 웹 앱에서 영수증 파일 업로드
        │
        ▼
[백엔드 서버 - FastAPI]
  ① 파일 수신 및 이미지 전처리
  ② Upstage Vision LLM 호출 → 가게명 / 날짜 / 품목 / 금액 추출
  ③ 구조화된 JSON 데이터로 변환 후 파일 저장
        │
        ▼
[데이터 저장]
  DB 없이 서버 로컬 JSON 파일(expenses.json)에 누적 저장
        │
        ▼
[배포]
  GitHub Push → Vercel 자동 배포
```

### 핵심 처리 포인트

- 영수증 이미지는 Base64로 인코딩하여 AI 모델에 전달
- AI 모델이 OCR + 데이터 구조화를 한 번에 처리
- 별도 DB 없이 JSON 파일로 운영하여 인프라 복잡도 최소화

---

## 6. 기대 효과


- **시간 절약**: 영수증 수동 입력 없이 AI가 자동 처리하여 기록 시간 최소화
- **소비 파악**: 날짜·품목별 지출 데이터를 시각적으로 확인
- **낮은 진입장벽**: 별도 앱 설치 없이 브라우저에서 즉시 사용 가능
- **빠른 배포**: 1일 개발 후 즉시 서비스 가능한 경량 구조
